# Haze’s Sprint Helper

A client-side Fabric mod for Minecraft Java Edition that automates sprinting and swimming with smooth, configurable, and performance-focused movement logic.

---

## ⚡ Features

### 🚀 Movement
- Auto Sprint while moving
- Optional omnidirectional sprint
- Auto Swim / water sprint support
- Smooth vanilla-like sprint behavior
- Sprint state caching (no spam toggling)

### 🌊 Water Movement
- Automatic swimming in water
- Works in still and flowing water
- Optional sneaking swim support
- Natural movement handling

### 🧠 Performance Systems
- Sprint debounce / cooldown system
- Prevents flickering
- Reduces packet spam
- Optimized tick handling

---

## 🎛️ Configuration

All settings are configurable in-game via Mod Menu or config GUI.

### General
- Auto Sprint (default: true)
- Omnidirectional Sprint (default: false)
- Sprint debounce system (default: true)

### Water Movement
- Auto Swim (default: true)
- Swim while sneaking (default: false)

### HUD
- HUD enabled
- Position & scale controls
- Sprint status indicator

### Keybinds
- Toggle Auto Sprint
- Open Config GUI

### Advanced
- Packet optimization
- Smooth sprint transitions
- Debug logging

## 🎮 Installation

1. Install Fabric Loader (compatible with your Minecraft version)
2. Install Fabric API (matching your version)
3. (Optional) Install Mod Menu for config UI
4. Place the mod jar into your `/mods` folder
5. Launch the game
